HookedJS
=====================

Is a Full-Stack typescript+graphql+passport+react framework.

Install
---
```
nvm use 12
npm i -g solidarity webpack typescript yarn
yarn lerna:bootstrap
```

Development
---
A local env can be launched by (1) starting Docker and (2) running 
```
yarn dev
```
